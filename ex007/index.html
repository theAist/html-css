Principais formatações de texto

Negrito/Destaque

Nesta frase, temos um termo em negrito usando a tag <b> (sem semântica).

Nesta frase, temos um termo em destaque usando a tag <strong> (com semântica).


Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (sem semântica).

Nesta frase, temos um termo com ênfase usando a tag <em> (com semântica).


Marcado

Podemos também criar um texto marcado usando a tag <mark style="">.

Também há como criar outro texto marcado com o <mark>.


Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.


Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.


Texto inderido

Podemos marcar um texto sublinhado usando a tag <u> (sem semântica)

Podemos marcar um texto como inserido com a tag <ins> (com semântica).


Texto sobrescrito/subscrito

Podemos inserir coisas do tipo x3+3.

Também inserir coisas do tipo H2O.
